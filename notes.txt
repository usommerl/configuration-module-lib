Logfile format
==============

General concept
---------------
<TIMESTAMP><SEPERATOR><USER><SEPERATOR><OPERATION><SEPERATOR>[VALUES]

Supported operations
--------------------
- Created symlink
  - Keyword : LNK
  - Values  : /home/example/lnkr_module/link_target<SEPARATOR>/home/example/link_location
- Created backup of file or folder
  - Keyword : BAK
  - Values  : /home/example/.bashrc<BACKUP_SUFFIX>

<TIMESTAMP>
-----------
YYYY-MM-DDThh:mm:ss[+-]hh:mm

<SEPERATOR>
-----------
\t\0

<BACKUP_SUFFIX>
---------------
.backup-<TIMESTAMP>
